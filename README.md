fungsi Program: Program ini menghitung volume tabung menggunakan rumus volume = π × r² × t, diimplementasikan dalam sebuah fungsi dengan parameter jari-jari (r) dan tinggi (t).
Cara Kerja Program:
Definisikan fungsi dengan parameter r (double) dan t (double), yang mengembalikan nilai volume.
Di dalam fungsi, hitung volume menggunakan Math.PI * r * r * t.
Di main method, panggil fungsi dengan input r dan t dari pengguna, lalu tampilkan hasil volume.
